# 行動規範

## 私たちの誓い

私たちはメンバー、コントリビューター、およびリーダーとして、年齢、体の大きさ、目に見えるまたは見えない障害、民族性、性の特徴、性自認および性表現、経験のレベル、教育、社会経済的地位、国籍、容貌、人種、宗教、または性的アイデンティティおよび指向に関係なく、すべての人にとってハラスメントのない体験を提供することを誓います。

私たちは、オープンで、歓迎的で、多様で、包括的で、健全なコミュニティに貢献する行動をとることを誓います。

## 私たちの標準

コミュニティにとって良い環境に貢献する行動の例：

- 他の人への共感とやさしさを示す
- 異なる意見、視点、経験を尊重する
- 建設的なフィードバックを与え、丁寧に受け入れる
- 自分の過ちの責任を受け入れ、影響を受けた人に謝罪し、その経験から学ぶ
- 個人としてだけでなく、コミュニティ全体に何が最善かを考える

許容できない行動の例：

- 性的な言葉や画像の使用、および性的な注目またはその他あらゆる種類の性的接近
- トローリング、侮辱的または軽蔑的なコメント、個人的または政治的な攻撃
- 公的または私的なハラスメント
- 明示的な許可なく、住所や電子メールアドレスなど、他者の個人情報を公開する
- 職業上不適切と合理的に考えられる他の行為

## 施行責任

コミュニティリーダーは、許容できる行動の標準を明確にし、施行する責任があり、不適切、脅威的、攻撃的、または有害と見なされる行動に対して適切かつ公正な是正措置をとります。

コミュニティリーダーは、この行動規範に沿っていないコメント、コミット、コード、Wiki の編集、Issue、およびその他のコントリビューションを削除、編集、または拒否する権利と責任を持ちます。また、モデレーションの決定に関する理由を適宜伝えます。

## 適用範囲

この行動規範は、すべてのコミュニティスペース内で適用されます。また、個人がパブリックスペースでコミュニティを公式に代表する場合にも適用されます。コミュニティを代表する例としては、公式の電子メールアドレスの使用、公式のソーシャルメディアアカウントを介した投稿、またはオンラインまたはオフラインイベントでの指定代理人としての行動などがあります。

## 施行

虐待的、嫌がらせ、またはその他の許容できない行動の事例は、GitHub の [Private Security Advisories](https://github.com/ikech4n/vulflare/security/advisories/new) または Issue にて報告することができます。すべての苦情は迅速かつ公正にレビューおよび調査されます。

すべてのコミュニティリーダーは、あらゆる事件の報告者のプライバシーと安全性を尊重する義務があります。

## 帰属

この行動規範は、[Contributor Covenant](https://www.contributor-covenant.org) バージョン 2.1 を参考に作成されています。
